# ela6105-latihan01
Multi page streamlit Praktikum Analisa Big Data Ekonomika.

## Siklus Tugas Kelompok
1. Masing-masing anggota kelompok menyiapkan Anaconda Prompt.
2. Ketik `conda activate prakbigdata`.
3. Ketik `cd \folder-kelompok`.
4. Ketik `jupyter lab`.
5. Buka Terminal di Jupyter lab.
6. Ketik di terminal `git pull`.
7. Edit file sesuai pembagian tugas di antara anggota kelompok.
8. Pastikan aplikasi Streamlit tidak error.
9. Ketik di terminal `git status`.
10. Ketik di terminal `git add NAMAFILE.py`.
11. Ketik di terminal `git status`.
12. Ketik di terminal `git push`. Jika ada warning, maka ketik dulu `git pull`, baru kemudian `git push`.
